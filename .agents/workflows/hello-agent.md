description: Un agente de ejemplo que dice hola y revisa el estado del proyecto.
---

Este es un workflow de ejemplo. Para ejecutar los comandos automáticamente sin preguntar, puedes usar la etiqueta `// turbo`.

1. Revisa el estado del repositorio git local.
// turbo
2. Crea un archivo temporal llamado `hola.txt` con un saludo.
3. Lee el archivo `hola.txt`.
4. Borra el archivo `hola.txt` después de leerlo.
